Choisir un récepteur prédéfini

Le choix d'un récepteur prédéfini est possible lors de l'application des deux méthodes. Pour se faire, créer un récepteur de type force motrice (méthode théorique). Puis dans son formulaire, cliquer sur le bouton Prédéfini.



Ou cliquer sur le bouton Prédéfini de l'onglet Editer un composant (méthode industrielle).



Ce qui nous permet d'afficher le catalogue des moteurs suivant :



Sélectionner un moteur puis cliquer sur le bouton OK.
